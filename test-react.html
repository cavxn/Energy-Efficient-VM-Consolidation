Troubleshooting

If React app is not working:

  1. Make sure Node.js is installed: node --version
  2. Install dependencies: npm install
  3. Start React app: npm start
  4. Check for errors in the terminal

Backend Services:

  1. Start Backend API: python vmp/api_server.py
  2. Start ML API: python vmp/ml_api_server.py
